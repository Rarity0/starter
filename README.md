# Starter
Use `startx` to start the UI. (It uses bspwm.)

> Minimal UI for Arch so you can install your needs more easily.

Checks which packages will be installed at here: [Package list](./packages.txt)

## Shortcuts
- `Ctrl+q`: Kitty terminal.
- `Super+1`: Workspace 1.
- `Super+shift+1`: Move window to other workspace.
- `Super+shift+q`: Close window.
- `Super+q`: Close task. (Force)
- `Super+l`: Lock screen.
- `Super+r`: Run panel (rofi).
- `Super+e`: File explorer (thunar).
